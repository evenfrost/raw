## Install

```bash
git clone git@github.com:evenfrost/raw.git
rm -rf .git
npm i
```
## Run

### development

```bash
npm start
```
Server is now up at [http://localhost:5000](http://localhost:5000).

### production

```bash
npm run build
npm run prod
```
Server is now up at [http://localhost:80](http://localhost:80).
